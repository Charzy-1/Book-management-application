# 📖 [Book_Management_Application] <a name="about-project"></a>

**[Book_Management_Application]** is a webpage designed to manage book entries. It has three pages: The homepage which has the list of books entered, the add-book page which has a form where the user can enter a book title and and author's name and then add book by clicking on the button and the third page which is the contact me page
